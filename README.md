Итоговый проект IT-Академии ТУСУР по программе «Python-разработчик».
Веб-приложение система учёта личных финансов.
Для запуска:
1. pip install -r requirments.txt
2. python manage.py makemigrations
3. python manage.py migrate
4. python manage.py runserver 8000
Веб-приложение будет доступно по адресу http://127.0.0.1:8000/
